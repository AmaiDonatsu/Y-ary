# Y-ary

A TypeScript package.

## Installation

```bash
pnpm add y-ary
```

## Usage

```typescript
import { hello } from 'y-ary';

console.log(hello('World'));
```

## Development

```bash
# Install dependencies
pnpm install

# Build the package
pnpm build

# Watch mode during development
pnpm dev
```
