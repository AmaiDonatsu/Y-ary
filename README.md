# Y-ary

A TypeScript package.

## Installation

### From npm registry

```bash
pnpm add y-ary
```

### From GitHub repository

Using **pnpm**:
```bash
pnpm add https://github.com/AmaiDonatsu/Y-ary.git
```

Using **npm**:
```bash
npm install https://github.com/AmaiDonatsu/Y-ary.git
```

## Usage

```typescript
import { hello } from 'y-ary';

console.log(hello('World'));
```

## Development

```bash
# Install dependencies
pnpm install

# Build the package
pnpm build

# Watch mode during development
pnpm dev
```
